IMAGE TAG(img)

--->HTML imag tag isused to dispaly image on the web paeg

---> HTML (img) tag is an empty tag that contains attributes only

ATTRIBUTES

  1. src

    --->The 'SRC' Attribute will specifies the path or (URL) to the IMAGE

    NOTE---> The image path should be in the same file location if not it will display 'alt' (it means Alternatigve Text)

  2. alt

    ---> The 'ALT' Attribute will specifies the 'Alternate text' for the image.

    NOTE---> If the browser cannot display the Image .then it will display the value of the 'Alrtenate Text'

  3. width and hightt

    --->The 'WIDTH & HIGHT' attribute always define the width and hight of IMAGE in pixels

    ---> Here we suggest using the 'STYLE' Attribute it will prevent sheet from changing the size of image.

    EXAMPLE:

    width='100' hight='100' (or) style=width:100px;hisht:100px;

  4. Aline=left/right

    --->This 'ALINE' is used to let the IMAGE to the rigt or left of thr page

    --->Aline is also calld as 'FLOAT'
